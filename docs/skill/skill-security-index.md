## ⚠️ 重要提示

### 不影响现有功能

| 功能 | 是否影响 | 说明 |
|------|---------|------|
| 编译 | ❌ 不影响 | 命令不变，只是内部多了混淆步骤 |
| 安装 | ❌ 不影响 | NSIS + install.ps1 完全不变 |
| 品牌化 | ❌ 不影响 | 改动完全独立 |
| Git 更新 | ❌ 不影响 | 代码完全不变 |
| 运行性能 | ❌ 几乎无影响 | 解密开销 < 1ms |

---

### 核心原则

1. **不改变用户体验**（无感知）
2. **不影响现有功能**（编译、安装、更新）
3. **分阶段实施**（先做简单的）
4. **可回滚**（出问题能快速恢复）
