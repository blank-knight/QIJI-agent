# 知识库模板（客户首次使用必读）

> 本文件是 GEO skill 的知识库引导模板。
> 客户首次使用时，skill 按此模板引导客户提供真实资料，整理后导入奇计网页端。

## 知识库类型

奇计网页端有两个知识库（图1红框标注）：

1. **企业画像图库** — 公司logo、产品图、团队照、环境图等
2. **企业知识库** — 文字资料，AI写文章的素材源

## 企业知识库模板

skill 引导客户逐项填写以下内容。措辞要求**通俗易懂，避免专业术语**（图3要求）。

```
【公司基本信息】
- 公司全称：
- 品牌简称/别名：
- 成立时间：
- 注册资本：
- 所属行业：
- 公司规模（人数）：
- 所在地区（省市区）：
- 官网地址：

【产品/服务】
- 主营产品或服务名称：
- 产品/服务核心卖点（3-5条）：
- 与竞品的核心差异：
- 典型应用场景：
- 价格区间（可选）：

【品牌方向】（默认全选，客户可删减）
□ 经验分享
□ 过程分享
□ 用户痛点
□ 用户避坑
□ 科技前沿
□ 行业洞察
□ 潮流趋势
□ 其他：（客户自定义）

【目标客户画像】
- 主要客户群体：
- 客户行业分布：
- 客户地域分布：
- 客户核心需求/痛点：

【竞争环境】
- 主要竞争对手（品牌名）：
- 竞争优势：
- 劣势或待改进点：

【资质荣誉】
- 行业认证：
- 专利数量：
- 获奖荣誉：
- 合作案例（可选，提供2-3个）：

【联系信息】
- 联系电话：
- 联系邮箱：
- 联系地址：
- 微信/其他社交账号（可选）：
```

## 企业画像图库清单

图片需要客户自己上传到网页端（skill 无法代操作图片上传）。

引导客户准备以下图片：

```
□ 公司LOGO（高清，透明背景最佳）
□ 产品图片（每个主营产品1-3张）
□ 团队/办公环境照
□ 生产车间/服务现场照（如有）
□ 资质证书扫描件
□ 合作客户案例配图（如有）
□ 其他品牌相关图片
```

## 导入流程

知识库资料整理完毕后：

1. **文字资料**：skill 整理成结构化文本 → 客户复制粘贴到网页端"企业知识库"
2. **图片资料**：客户手动上传到网页端"企业画像图库"
3. **验证**：导入后用 `knowledge` 命令查看是否成功

## 强约束

⚠️ **知识库必须基于真实资料，不能编造。**

原因：
- 编造内容会导致 GEO 效果反向（AI 搜索引擎引用了错误信息）
- 客户的潜在客户被误导，损害品牌信誉
- 各AI平台的内容质量审核可能封禁虚假内容

skill 的职责是**引导整理**，不是**替客编造**。如果客户提供的信息不完整，skill 应：
1. 明确指出缺什么
2. 用对话引导补充
3. 绝不自行编造填补空白
