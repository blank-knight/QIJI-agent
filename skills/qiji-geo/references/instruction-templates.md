# 写作指令模板（12平台 × 3类）

> 本文件是 GEO skill 的指令配置引导模板。
> 每个平台必须有3类指令：文章创作 / 标题创作 / 流量复刻。
> skill 根据这些默认模板引导客户定制，确认后导入网页端"写作指令"。

## 三类指令说明

| 类型 | 用途 | 输出 |
|------|------|------|
| 文章创作 | 生成完整文章正文 | 800-2000字文章 |
| 标题创作 | 生成文章标题 | 25-30字标题 |
| 流量复刻 | 基于爆文模板改写 | 仿写文章 |

## 默认指令模板

### 一、文章创作指令（通用基础版）

```
你是一位专业的{行业}领域内容创作者。

请根据以下要求创作文章：
1. 主题：{关键词}
2. 字数：800-1500字
3. 视角：站在{公司名}的角度，展示专业性和行业洞察
4. 风格：专业但不枯燥，有数据支撑，有实际案例
5. 结构：标题 → 引言 → 正文（3-4个观点）→ 总结
6. 要求：
   - 自然融入品牌名「{品牌名}」
   - 不使用夸张表述（如"最好""第一""唯一"）
   - 每个观点用具体案例或数据支撑
   - 结尾引导关注或咨询

参考知识库：{知识库内容摘要}
```

```
请根据以下要求为文章创作标题：
1. 文章主题：{关键词}
2. 字数：25-30字

   - 准确反映文章核心内容
   - 有节奏感，读起来顺口
4. 数量：生成5个备选标题

```

### 三、流量复刻指令（通用基础版）

```
你是一位专业的爆文仿写师。

请根据以下原文进行改写创作：
1. 原文：{爆文URL或内容}
2. 要求：
   - 保持原文的核心观点和结构
   - 替换为{品牌名}的行业案例
   - 调整语气适配{目标平台}的内容风格
   - 字数与原文相当
   - 去除原文中的竞品信息
   - 自然融入{品牌名}和知识库内容
3. 输出：改写后的完整文章

参考知识库：{知识库内容摘要}
```

## 引导流程

skill 触发指令配置时：

1. 展示12个平台清单，询问客户需要配置哪些平台（默认全选）
2. 对每个选中的平台：
   - 展示该平台的3类默认指令
   - 询问客户是否需要调整
   - 确认后记录
3. 所有平台确认后：
   - 生成完整指令清单
   - 引导客户导入网页端"写作指令"页面
4. 验证：用 `instructions` 命令查看是否导入成功

## 强约束

⚠️ **每个平台每类指令都必须有，缺一个该平台就不能正常工作。**

原因：
- 文章创作指令缺失 → AI不知道怎么写文章
- 标题创作指令缺失 → AI生成的标题不符合平台调性
- 流量复刻指令缺失 → 爆文复刻无法正常改写
